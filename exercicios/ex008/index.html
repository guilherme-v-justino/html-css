Principais formatações

Negrito

Nessa frase temos um termo em negrito utilizando a tag <b>

Nesta frase temos um termo em destaque usando a tag <strong>

Itálico

Nesta frase temos um termo em itálico usando a tag <i>

Nesta frase temos um termo com ênfase utilizando a tag <em>

Texto marcado

Podemos criar também um texto marcado usando a tag <mark>

E no outro parágrafo temos outro texto marcado

Texto grande e pequeno

Estamos criando um texto grande com a tag (não semântica) <big> e um texto pequeno com a tag (semântica) <small> nessa frase

Texto deletado

Podemos marcar um texto para que ele deve ser lido com a tag <del>, mas não considerado

Texto inserido

Podemos marcar um texto como incluído com a tag <ins>, para demonstrar que ele foi adicionado depois

Existe também o sublinhado com a tag <u>, porém não é semântica

Texto sobrescrito

Para inserir coisas tipo x20 + 3, com a tag <sup>

Texto subscrito

Para inserir coisas tipo H20, com a tag <sub>
